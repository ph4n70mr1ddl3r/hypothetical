## 2. Purpose and Scope

### 2.1 Purpose

This guide provides step-by-step instructions for all BuildRight associates on how to use Oracle Fusion Cloud ERP in their daily work. It covers:

- **System Access**: How to log in and navigate the Oracle interface
- **Common Tasks**: Role-based procedures for store associates, managers, and corporate users
- **Mobile Access**: Using the Oracle Mobile App for on-the-go tasks
- **AI Features**: How to interact with Oracle AI agents and digital assistant
- **Getting Help**: Support resources and troubleshooting

### 2.2 Scope

This guide applies to:

- All 503,000 associates across North America
- All Oracle Fusion Cloud ERP modules (Phase 1-3)
- All devices (desktop, tablet, mobile)

### 2.3 Guiding Principles

1. **User-Centric Design**: Simple, intuitive instructions for non-technical users
2. **Role-Based Content**: Tailored sections for each job role
3. **Just-in-Time Learning**: Quick references for common tasks
4. **AI Integration**: Guidance on leveraging AI for productivity

## 3. Getting Started

### 3.1 System Access

#### 3.1.1 Desktop Access

1. **Open Browser**: Use Chrome, Edge, or Safari (recommended)
2. **Navigate to**: `https://erp.buildright.com`
3. **Single Sign-On (SSO)**: Use your existing BuildRight network credentials
4. **Multi-Factor Authentication (MFA)**: Complete MFA via Microsoft Authenticator or SMS
5. **Landing Page**: You will see your role-specific dashboard

#### 3.1.2 Mobile Access

1. **Download**: Oracle Mobile App from Apple App Store or Google Play Store
2. **Server URL**: `erp.buildright.com`
3. **Authentication**: Same SSO and MFA as desktop
4. **Biometric Login**: Enable Face ID/Touch ID for faster access

### 3.2 Navigation Basics

**Main Navigation Menu:**

| Icon | Module | Description |
|---|---|---|
| 🏠 | Home | Role-based dashboard with key metrics and tasks |
| 📊 | Financials | General Ledger, Accounts Payable, Accounts Receivable |
| 🛒 | Procurement | Purchase Requisitions, Purchase Orders, Supplier Management |
| 📦 | Supply Chain | Inventory, Order Management, Logistics |
| 👥 | HCM | Human Capital Management, Payroll, Learning |
| 📈 | EPM | Enterprise Performance Management, Planning |
| 🛡️ | Risk Management | Controls, Compliance, Monitoring |
| 🔍 | Search | Global search across all modules |
| 🤖 | Digital Assistant | AI-powered help and transaction assistance |

**Key Navigation Tips:**
- Use the **Global Search** (Ctrl+K) to find any transaction, report, or help
- **Personalize** your homepage by adding/removing tiles
- **Watchlist** allows you to track key items (approvals, alerts, favorites)
- **Notifications** (bell icon) show pending tasks and system alerts

## 4. Role-Based Quick Start Guides

### 4.1 Store Associate (Sales Associate, Cashier)

**Daily Tasks:**

| Task | Oracle Module | Steps |
|---|---|---|
| **Check Inventory** | SCM Cloud - Inventory | 1. Open Oracle Mobile App → Inventory<br>2. Scan item barcode or enter SKU<br>3. View on-hand quantity across stores |
| **Process Return** | Financials Cloud - Receivables | 1. At POS, initiate return<br>2. Oracle validates return policy<br>3. Process refund/exchange |
| **Look Up Customer** | CX Cloud - Sales | 1. Open Customer Search<br>2. Enter phone number or loyalty ID<br>3. View purchase history and preferences |
| **Price Check** | SCM Cloud - Pricing | 1. Scan item barcode<br>2. View current price, promotions, Pro pricing |
| **Submit Time Card** | HCM Cloud - Time & Labor | 1. Open Timecard<br>2. Enter hours worked<br>3. Submit for approval |

**AI Assistance:**
- Ask the Digital Assistant: "What's the inventory for SKU 12345?"
- Use voice command: "Hey Oracle, find the nearest store with plywood in stock"

### 4.2 Store Manager

**Daily Tasks:**

| Task | Oracle Module | Steps |
|---|---|---|
| **View Store P&L** | Financials Cloud - GL | 1. Open Financial Dashboard<br>2. Select store P&L report<br>3. Drill down to department level |
| **Approve Purchase Requisition** | Procurement Cloud | 1. Open Approvals → Purchase Requisitions<br>2. Review items, quantities, budget<br>3. Approve/Reject with comments |
| **Review AI Forecast** | SCM Cloud - Demand Planning | 1. Open Demand Forecast dashboard<br>2. View AI-generated forecasts<br>3. Adjust overrides if needed |
| **Run Payroll Report** | HCM Cloud - Payroll | 1. Open Payroll → Reports<br>2. Select pay period<br>3. Generate department payroll summary |
| **Approve Employee Time** | HCM Cloud - Time & Labor | 1. Open Timecard Approvals<br>2. Review team timecards<br>3. Approve/Reject with comments |

### 4.3 Corporate User (Finance, Procurement, HR)

**Key Transactions:**

| Function | Oracle Module | Transaction |
|---|---|---|
| **Create Journal Entry** | Financials Cloud - GL | `Journals → Create` |
| **Process Invoice** | Financials Cloud - AP | `Invoices → Create` |
| **Create Purchase Order** | Procurement Cloud | `Purchase Orders → Create` |
| **Run Financial Report** | EPM Cloud | `Reports → Financial Reporting Studio` |
| **Process Payroll** | HCM Cloud - Payroll | `Payroll → Submit Payroll` |

## 5. AI and Digital Assistant

### 5.1 Digital Assistant Overview

The Oracle Digital Assistant is an AI-powered chatbot that helps you complete tasks, find information, and get answers.

**Access:**
- Desktop: Click the 🤖 icon in the top-right corner
- Mobile: Tap the chat bubble icon

**Example Queries:**

| Query Type | Example |
|---|---|
| **Transaction Help** | "How do I create a purchase requisition?" |
| **Data Lookup** | "What's the balance for customer ABC?" |
| **Process Guidance** | "What's the approval limit for store supplies?" |
| **Report Generation** | "Generate a report of overdue invoices" |
| **System Status** | "Is the system down for maintenance?" |

### 5.2 AI Agents Interaction

**Financial Agents:**
- **Expenses Agent**: Submit expense reports by uploading receipts
- **Payables Agent**: Ask about invoice status or payment scheduling

**Procurement Agents:**
- **Sourcing Agent**: Get supplier recommendations
- **Contract Agent**: Review contract terms and risks

**SCM Agents:**
- **Demand Forecasting Agent**: View AI demand predictions
- **Inventory Optimization Agent**: Get stock level recommendations

**HCM Agents:**
- **Learning Agent**: Get personalized training recommendations
- **Engagement Agent**: Participate in pulse surveys

## 7. Reporting and Analytics

### 7.1 Oracle Analytics Cloud

**Access**: Desktop only via `https://analytics.buildright.com`

**Key Reports:**

| Report | Audience | Description |
|---|---|---|
| **Store P&L Dashboard** | Store Managers | Daily sales, margin, expense metrics |
| **Inventory Health Dashboard** | Merchandising | Stock levels, turns, shrink |
| **Customer Insights** | Marketing | Purchase patterns, loyalty metrics |
| **AI Forecast Accuracy** | Supply Chain | Forecast vs. actual performance |

### 7.2 Self-Service Reporting

**Creating a Simple Report:**
1. Open Oracle Analytics → New → Report
2. Select data source (e.g., "Sales Transactions")
3. Drag and drop columns
4. Apply filters
5. Save and share

**Scheduling Reports:**
1. Open any report → Schedule
2. Set frequency (daily, weekly, monthly)
3. Choose recipients
4. Save schedule

## 10. Tips and Best Practices

### 10.1 Productivity Tips

1. **Use Keyboard Shortcuts**:
   - `Ctrl+K`: Global Search
   - `Ctrl+S`: Save current transaction
   - `Ctrl+P`: Print current page
   - `F1`: Help for current screen

2. **Personalize Your Dashboard**:
   - Add tiles for frequently used transactions
   - Remove tiles you don't need
   - Set default landing page

3. **Leverage Watchlist**:
   - Add items to track (e.g., pending approvals)
   - Set alerts for threshold breaches

4. **Use Digital Assistant First**:
   - Ask before searching manually
   - Use natural language queries

### 10.2 Common Issues and Solutions

| Issue | Solution |
|---|---|
| **"No Access" error** | Contact manager to verify role assignment |
| **Slow performance** | Clear browser cache, try incognito mode |
| **MFA not working** | Ensure time on device is correct; use backup code |
| **Data not updating** | Refresh page; check internet connection |
| **AI recommendation wrong** | Provide feedback via thumbs down; report to CoE |
